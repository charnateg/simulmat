# simulmat

Real-time cloth physics simulator built with Three.js and TypeScript.

Configure material type, physics parameters (gravity, wind, mass, stiffness…), and lighting — then interact with the cloth using tools: hand, knife, fire, push, bullet.

## Stack

- **Three.js** — 3D rendering
- **TypeScript** + **Vite** — build

## Run

```bash
npm install
npm run dev
```

## Presets

Simulation state can be saved and shared via URL hash (auto-encoded snapshot).
